Basic Rules Basic Rules
  • The ball must be within the boundary lines at all times and if a player hits the ball out of bounds, the opponent gets the point
    • If the ball touches the boundary line, that is still considered in bounds
  • There are 4 points per game and they increment from Love(0)-15-30-40-Game
    • Everyone starts at Love(0)
    • Winner of the game is the one that reaches game
  • Players must allow the serve to bounce first before returning it
  • The server serves for the whole game and switches after each game
